Projet 1 : SignalHire — Assistant de Carrière Intelligent & Radar Sourcing

Développement d'une plateforme SaaS complète permettant d'automatiser la recherche d'emploi et la personnalisation stratégique des candidatures grâce aux Large Language Models (LLMs).

Piliers Technologiques

  • Radar Sourcing : Agrégation d'offres en temps réel via APIs (Welcome to the Jungle, Google Jobs, France Travail, Adzuna).
  • Scoring Prédictif : Analyse sémantique de compatibilité entre offres et Master CV via Gemini et Groq.
  • Automatisation LaTeX : Génération dynamique de CV et lettres de motivation optimisés pour les ATS.

Architecture & Robustesse

  • Tiers de Fallback : Système de rotation de clés API et bascule autonome en cas de saturation des quotas LLM.
  • Stack Moderne : Backend FastAPI, Frontend Next.js et base de données SQLite versionnée.
  • Repo Privé - Accès sur demande

Compétences Mobiles : Python (FastAPI) TypeScript (Next.js) Generative AI (LLMs) Data Aggregation LaTeX Web Automation Architecture SaaS
